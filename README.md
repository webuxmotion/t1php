# T1PHP - Web Framework

## First TRY:

### 1. Run docker containers:
```
docker-compose up -d
```

### 2. Install composer:
```
composer install
```

### 3. Import DB

1. Go to http://localhost:40002/

2. Use username: *t1phpuser* and password: *t1phppass* to login

3. Import DB.sql

### 4. Install node_modules
```
yarn
```

### 5. Run webpack
```
yarn watch
```
